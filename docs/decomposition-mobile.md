# Functional Decomposition - Hunted Mobile (MVP)

## 1. Access / Authentication (IAM)
### 1.1. Sign In
- Login form
- "Forgot password" link
- Social login options (Google, LinkedIn, GitHub)

### 1.2. Sign Up
- Registration by email
- Social login options
- Role selection (Recruiter / Candidate)
- Redirect to profile creation

### 1.3. Session Management
- Stay logged in
- Logout option

## 2. Profile (MVP Subset)
- View profile
- Edit profile options
- Switch between recruiter/candidate profiles
- Connect social networks (LinkedIn, GitHub, Google)
- Change password
- Activate/deactivate candidate profile

## 3. Chats (MVP Subset)
### 3.1. Chats List
- View active chats
- Archive chat option
- Mark chat as favourite

### 3.2. Chat Actions
- Send and receive messages
- Recruiter contact restriction (Hidden until candidate allows)
- Candidate control (Open/decline sharing contact details)
- Recruiter: Send offer
- Recruiter: Mark chat as rejected

## 4. Navigation / UI
- Bottom navigation bar
- Chats access
- Profile access
- Settings access
- App settings
